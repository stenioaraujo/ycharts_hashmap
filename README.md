## Testing

``` bash
git clone https://github.com/stenioaraujo/ycharts_hashmap
cd ycharts_hashmap
pip install pipenv
pipenv install --dev
pipenv run tox
```

## Install locally

``` bash
git clone https://github.com/stenioaraujo/ycharts_hashmap
cd ycharts_hashmap
pip install pipenv
pipenv run pip install .
```
